# Brand voice directory — Notion

> **Real company, public data.** Extracted from Notion's **public** marketing copy (see
> [`../input.md`](../input.md) for the harvest and sources). Samples are verbatim public copy. This is
> a demonstration of the `brand-voice-guide` skill's output, not an official Notion document.

Presented with the *same file structure* as the Linear example on purpose — read the two side by side
to see the skill capturing two clearly different voices in the same category.

| File | What it holds |
|---|---|
| `voice-core.md` | The character — tonal pillars (each with a verbatim sample), sentence rules, persona-we-are-NOT, the distinctiveness test |
| `writing-standard.md` | The concrete rules — a *Do* half and a severity-tagged *Avoid* half |
| `voice-website.md` | The one intentional channel harvested (marketing site / product pages) |
| `about-company.md` | Light voice context: what it is, the stance |
| `icp.md` | Light reader context (replace with `icp-research` output if you have it) |
| `real-examples.md` | In-voice lines to reference for shape |

## Side-by-side: why Linear and Notion read as different voices
Same category, same AI-era moment, opposite registers. **Linear** strips to the plainest true statement
and trusts silence (*"Built for the future. Available today."*). **Notion** warms the same idea and
talks to you (*"Less tracking. More progress."*). The skill's job is to name *that* difference — the
thing that would be lost if either brand wrote the other's copy.
